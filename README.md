# Agentic Governance Reference

An interactive reference architecture for agentic claims processing governance, grounded in the Allianz Partners / Otera transformation case. The tool covers the four core governance design problems that any organisation deploying autonomous agents at scale must solve independently of the platform they use: authority (what each agent may decide), accountability (who owns outcomes), change (how the system evolves), and data and decision quality (what counts as sufficient explanation). Structured across nine tabs — process flow, agent register, governance layers, operating model, governance flow, platform vs. org design, regulatory obligations, and compliance reference — it is intended as a concrete, explorable model rather than an abstract framework.

Built with Vite + React + TypeScript.
